# my-first-repo

A beginner repository to learn Git, GitHub, Actions, and GitHub Copilot.

## What's here

- Basic repo setup and docs
- Minimal Python example and tests
- GitHub Actions workflow to run tests

## How to use

1. Clone: `git clone https://github.com/YOUR-USERNAME/my-first-repo.git`
2. Create a branch: `git checkout -b feature/your-name`
3. Make changes, commit, push, open a PR on GitHub
